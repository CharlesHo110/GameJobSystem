Detailed Description

template<typename PT, typename T>
struct vgjs::awaitable_coro< PT, T >

Awaiter for awaiting a Coro of type Coro<T> or std::function<void(void)>, or std::pmr::vector thereof.

The caller will await the completion of the Coro(s). Afterwards, the return values can be retrieved by calling get() for Coro<t>.
