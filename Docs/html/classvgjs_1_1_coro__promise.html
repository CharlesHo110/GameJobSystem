Detailed Description

template<typename T = void>
class vgjs::Coro_promise< T >

Promise of the Coro. Depends on the return type.

The Coro promise can hold values that are produced by the Coro. They can be retrieved later by calling get() on the Coro (which calls get() on the promise)
